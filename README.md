
 Projeto FastAPI   SQLAlchemy  Pydantic  Uvicorn  Postgres

Um projeto desenvolvido com FastAPI https://fastapi.tiangolo.com/ para criação de APIs de forma rápida e eficiente.  

- Funcionalidades:  
CRUD de Usuários: Implementação de operações de criação, leitura, atualização e exclusão de usuários, permitindo o gerenciamento completo dos dados dos usuários.

Autenticação com JWT: Uso de JSON Web Tokens para autenticar e autorizar usuários, garantindo a segurança das operações e o acesso controlado aos recursos da API.

Integração com Banco de Dados: Utilização do SQLAlchemy para interagir com o banco de dados, facilitando a manipulação e persistência dos dados de forma eficiente.

Validação de Dados com Pydantic: Emprego do Pydantic para validar e serializar os dados de entrada e saída, assegurando a integridade e a conformidade dos dados processados pela API.

Documentação Automática da API: Aproveitamento dos recursos do FastAPI para gerar automaticamente a documentação interativa da API, acessível através do Swagger UI e ReDoc, facilitando o entendimento e a interação com os endpoints disponíveis.

- Tecnologias Utilizadas:
  
- Python 3.9+  
- FastAPI  
- SQLAlchemy  
- Pydantic  
- Uvicorn
